# How to use

## Layout

- `backend/` - Replicator core packages + tooling
- `frontend/` - (not used)
- `infra/` - (not used)

## Run locally

1. `cd backend`
2. Install dependencies:
   - `npm ci` (recommended)
3. See `backend/README` or package scripts in `backend/package.json`.

## Deploy

This repo is primarily a tooling/library repo. If you deploy components, document them under `infra/`.
